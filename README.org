To use this configuration one must:

- run ~git clone git@github.com:rvf0068/emacs.d.git .emacs.d~
- create a blank file called ~diary~ in the home directory, and another
  called ~custom.el~ in the directory ~.emacs.d~.
- create a directory =~/backups= in the home directory
- go to =~/.emacs.d/site-lisp/org-mode= and run =make= (In ubuntu, one has
  to install the package =texinfo=).
